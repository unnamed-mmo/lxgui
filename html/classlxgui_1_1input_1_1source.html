Detailed Description

The base class for input source implementation.

Note
In case you want to share the same source for multiple input::manager, you have to call set_manually_updated(true), and update the source yourself. Else, it will be updated by each input::manager (which may not be desirable).

◆ toggle_mouse_grab()

virtual void lxgui::input::source::toggle_mouse_grab ( )
pure virtual

Toggles mouse grab.

When the mouse is grabbed, it is confined to the borders of the main window. The actual cursor behavior when reaching those borders is of no importance : what matters is that relative mouse movement is always aquired, i.e. the mouse is never blocked. The mouse is not grabbed by default.
