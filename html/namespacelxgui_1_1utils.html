◆ variant

using lxgui::utils::variant = typedef std::variant< empty, bool, std::int64_t, std::int32_t, std::int16_t, std::int8_t, std::uint64_t, std::uint32_t, std::uint16_t, std::uint8_t, double, float, std::string>

Type-erased value for passing arguments to events.

Note
This contains all the basic numerical types up to 64bit precision. Any other scalar types (such as charXX_t, or strongly typed enums) can be represented as one of these. Use utils::get() to enable automatic support for strongly typed enums.
The utils::empty type is used to indicate an un-specified value. This is also the state of a default constructed utils::variant. When transferred to Lua, utils::empty is translated as nil.
